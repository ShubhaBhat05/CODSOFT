# CODSOFT

This is a simple Java program that simulates an ATM machine.
The program allows multiple users to log in using account number and PIN and perform basic banking operations like deposit, withdrawal, and balance checking.

# Programming features used
Java Basics: Variables, data types, operators  
Loops: while loop for continuous operations  
Conditional Statements: if-else for validation and menu handling  
Scanner Class: To take input from the user  
ArrayList: To store multiple user accounts and transaction history  
OOP Concepts: Classes and objects (ATM, BankAccount)  
Console I/O: Interaction with the user through command-line  

# How to run
Open ATMSystemReal.java in any Java IDE or editor.  
Run the program.  
Create accounts and log in using account number and PIN.  
Choose options from the menu to perform operations.
