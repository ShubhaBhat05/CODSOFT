# CODSOFT
This is a simple Java program that lets the user play a number guessing game.  
The program generates a random number between 1 and 100, and the user tries to guess it.  
After each guess, the program tells the user whether the guess is too high, too low, or correct.

# Programming features used  
Java Basics: Variables, data types, operators  
 Loops: while loop for multiple guesses  
 Conditional Statements: if-else for checking guesses  
 Scanner Class: To take input from the user  
 Random Class: To generate a random number  
 Oops Concepts: Use of classes and main method  
 Console I/O: Interaction with the user through command-line

# How to run
1. Open `Numberguessinggame.java` in  any Java IDE or editor.  
2. Run the program.  
3. Enter your guesses in the console when prompted.
