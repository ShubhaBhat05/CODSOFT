# Studdent grade calculator
This Java program calculates a student's grade based on marks entered for different subjects.  
It computes total marks, percentage, and assigns a grade based on the percentage.

# Programming Features Used
   Java Basics : Variables, data types, arithmetic operations  ,loops, conditional Statements
   OOP: Use of classes and methods  
   Console I/O: Interaction with the user via command-line
   
# How to run
1. Open `StudentGradeCalculator.java` in any Java IDE or editor.  
2. Run the program.  
3. Enter marks for the subjects when prompted.  
4. The program will display the total, percentage, and grade.
